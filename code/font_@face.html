    　2015年是世界反法西斯战争暨中国抗日战争胜利70周年。中国人民抗日战争是世界反法西斯战争的重要组成部分，作为世界反法西斯战争的东方主战场，中国战场爆发时间最早、历时最长。中国人民抗日战争坚定了盟国与法西斯作战的信心，推动了世界反法西斯统一战线的形成，中国人民也为此付出了巨大的民族牺牲。 　　为了铭记历史、缅怀先烈、珍视和平、开创未来，人民网强国论坛特别推出“世界反法西斯战争暨中国抗日战争胜利70周年”系列访谈活动，希望唤起每一个善良的人对和平的向往和坚守，避免历史悲剧重演，共同捍卫二战胜利果实，开创人类更加美好的未来。
